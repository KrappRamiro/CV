# CV – Ramiro Krapp

**[View CV](https://krappramiro.github.io/CV/krapp_ramiro_cv.pdf)**

Built with [Typst](https://typst.app/).

## Usage

Watch for live changes:
```bash
just watch
```

Compile once:
```bash
just build
```

Output: `krapp_ramiro_cv.pdf`

## Structure

```
├── main.typ            # Template
├── configuration.yaml  # Content (name, jobs, education, skills)
├── settings.yaml       # Fonts and spacing
└── fonts/
```
